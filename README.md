# how to setup 
`git clone https://github.com/RayenMnif/nvim-configs.git .config/nvim`

this configuration uses [Packer](https://github.com/wbthomason/packer.nvim) as a package manager so type `:PackerInstall` when you first load nvim or you'll get errors cause the plugins aren't downloaded
